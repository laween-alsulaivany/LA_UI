12. Utilities: utilities.css

Small single-purpose helpers fill gaps between components. Each class does one thing.

Keep utilities small

Each helper should do one thing. If a pattern repeats across several components, promote it to a component. Do not use utilities to recreate component behavior.

Do not chain layout utilities

Stacking .la-grid-3 inside .la-split inside .la-flex-row is a sign that you need a proper component, not more helpers.

CSS grid helpers for common column counts. All collapse to a single column at 720px. Use .la-grid-auto when the item count is unknown. It wraps automatically. Use .la-split for a content area plus a fixed 330px side panel.
