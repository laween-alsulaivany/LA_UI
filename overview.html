LA UI Design System

This CSS style guide documents the approved patterns for app UI, especially Django apps. It shows the components, usage rules, and markup developers should copy.

Use this as the shared UI library for apps. Change client branding through tokens. Keep component markup and class names stable unless the system needs a new component or style class.

How to use this

  • For a new client, change token values and product text first.
  • Pick the component that already exists here.
  • Do not create page-only variants unless the pattern belongs in the system.
