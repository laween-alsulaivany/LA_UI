04. Components: navigation.css

Navigation patterns tell the user where they are, where they can go, and which view is active. Pick the right pattern for the context.

Active state

Use is-active for UI state driven by JavaScript. Use aria-current="page" for the current page link in server-rendered HTML. Both apply the same styles.

Nav link variants

Pick one type per nav: code label, icon, or plain. Do not mix all three in the same nav. Use .la-nav-link--plain when there is nothing to show in the left slot.

Sidebar nav

Built from .la-nav-section groups. Each group has an optional .la-nav-title and a .la-nav stack of links. Three left-slot variants: code, icon, or plain.

Breadcrumbs

Two patterns. .la-breadcrumbs is for the topbar, shows parent context inline. .la-breadcrumb-trail is a page-level trail with links and aria-current="page" on the last item.

Tabs

Use to switch between related views of the same content. Do not use tabs for page-to-page navigation, use the sidebar nav for that.

.la-tabs .la-tab-btn

Pagination

Place pagination below a table or list. The .la-pagination-info label is optional but recommended when the total count matters to the user.

.la-pagination .la-page-link.is-current

Table switcher

Use to switch between related table views without leaving the page. The .la-table-switcher-sep separator can cluster links into logical groups.
